### Concept

Potion Master is a gesture-controlled game where players:

* Use real-time hand tracking

* Pinch to grab potion bottles

* Pour ingredients into a cauldron

* Trigger magical reactions

The goal is to experiment with potion combinations and create different magical outcomes.

### Current Features

* Real-time hand tracking

* Shelf-based potion layout

* Drag & snap system

* Pour animation sequence

* Cauldron stir animation

* Depth-based rendering system
